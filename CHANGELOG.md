## [v0.2.0] - 2025-01-01

### ✨ 新增功能
- **🎨 UI 全面升级**：采用霓虹灯特效和玻璃磨砂设计风格
- **🔧 折叠式侧边栏**：可收缩的历史记录和设置面板，点击即可开关
- **🌈 视觉特效增强**：
  - 霓虹光晕动画效果
  - 玻璃质感的毛玻璃效果
  - 流畅的渐变色彩过渡
  - 响应式卡片化布局
- **📱 交互体验优化**：
  - 点击面板外部自动关闭侧边栏
  - 更直观的设置界面
  - 优化的搜索体验
- **🚀 部署方式扩展**：
  - 完整的 Docker 支持（Docker Compose + 单容器）
  - Vercel 一键部署优化
  - Cloudflare Pages 部署指南
  - Netlify 和 Render 平台支持

### 🔧 技术改进
